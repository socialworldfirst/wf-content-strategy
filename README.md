# WorldFirst Content Strategy

Social-first content strategy for WorldFirst — four content pillars, the format system, and browsable platform-by-platform mockups (TikTok, Instagram, YouTube, LinkedIn, WhatsApp Channel).

**Live:** https://socialworldfirst.github.io/wf-content-strategy/ (password `wf`)

## Build

```bash
pip install cryptography
python3 build.py
```

Encrypts the inner content with PBKDF2-SHA256 + AES-GCM. Password: `wf`.
